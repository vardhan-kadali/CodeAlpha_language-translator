# CodeAlpha_language-translator
 Language Translation Tool

A simple frontend web application that translates English text into multiple languages and provides text-to-speech output for the translated text.

This project is implemented using HTML with embedded CSS and JavaScript, and it runs directly in the browser without any installation.

Project Description

The Language Translation Tool allows users to enter English text, translate it into a selected target language, and listen to the translated text using the browser’s built-in speech synthesis feature.
All styling and logic are included within a single HTML file for simplicity and beginner-friendly understanding.

 Features

Translate English text into multiple languages

Clean and responsive user interface

Text-to-speech for translated output

No backend or server required

Runs directly in the web browser

 Technologies Used

HTML – Structure of the web page

CSS (embedded) – Styling and layout of the interface

JavaScript (embedded) – Translation logic, API calls, and speech functionality

MyMemory Translation API – Language translation

Web Speech API – Text-to-speech functionality

 How to Run the Project

Download or clone this repository

Open the index.html file in any modern web browser

Enter English text in the input box

Select the target language

Click Translate to see the translated text

Click Speak to hear the translated output

No additional setup or installation is required.
